Overview

slender.css is a progressive CSS framework with following features.

  • Only 1 CSS file.

    • No JavaScript
    • No image file

  • Semantic markup.

  • Using progressive HTML5 and CSS3 features:

    • CSS variables
    • hsl()
    • calc()
    • <dialog> element
    • selector :invalid
    • pseudo-class :is()

  • Customizable colors, sizes and properties overwriting pre-defined CSS variables.

  • Size is automatically adjusted by base font size.

    • padding
    • line-height
    • border-radius

  • No layout supports. This framework is just a simple set of elements without margins.

Download the latest version from here. See activity on GitHub.

Customize

Customize overwriting following CSS variables.

--hHue of primary color.
--sSaturation of primary color.
--lLightness of primary color.
--bg-hHue of background color.
--bg-sSaturation of background color.
--bg-lLightness of background color.
--radiusBorder radius. Unit should be em.
--durationAnimation duration.
--(color)

Colors.

--white --light --gray-lighter --gray-light
--gray --gray-dark --dark --black
--primary --secondary --success --info
--warning --danger --blue --indigo
--purple --pink --red --orange
--yellow --green --teal --cyan 

              
                :root {

                  --h: 211;

                  --s: .8;

                  --l: .95;

                  --bg-h: 211;

                  --bg-s: .30;

                  --bg-l: .14;

                  --radius: calc(1em / 5);

                  --duration: .25s;

                }
